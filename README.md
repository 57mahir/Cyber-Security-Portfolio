# Cyber-Security-Portfolio


These reports demonstrate the practical application of cyber security principles, ranging from offensive security and malware reverse-engineering to defending critical cyber-physical infrastructure.

---

## 1. Enterprise Penetration Testing and Vulnerability Assessment
* **Objective:** Conducted comprehensive offensive security testing on simulated enterprise environments to identify, exploit, and document critical system vulnerabilities.
* **Methodology:** Utilised industry-standard reconnaissance, scanning, and exploitation techniques to assess the external and internal attack surface.
* **Key Outcomes:** Produced a detailed technical report outlining identified attack vectors, exploited flaws, and actionable remediation strategies to harden the infrastructure against future attacks.


## 2. Malware Analysis and Threat Intelligence
* **Objective:** Performed static and dynamic analysis on malicious files within a secure, isolated sandbox environment to understand execution flows and system impact.
* **Methodology:** Reverse-engineered malware behaviour to extract critical Indicators of Compromise (IoCs) and identify specific attack methodologies.
* **Key Outcomes:** Authored a comprehensive threat intelligence report detailing the malware's functionality, persistence mechanisms, and exact mitigation procedures for enterprise environments.


## 3. Cyber-Physical Systems Security Assessment
* **Objective:** Analysed the security architecture of interconnected cyber-physical systems, assessing vulnerabilities across both Operational Technology (OT) and IT networks.
* **Methodology:** Developed structured threat models to evaluate the physical and digital risks associated with smart infrastructure and interconnected hardware.
* **Key Outcomes:** Proposed robust, scalable security controls and remediation frameworks designed to protect critical physical infrastructure from logical attacks and unauthorised access.
